# titanic-survivors-ml

This is a machine learning project to predict Titanic passenger survival according age,gender,Pclass,cabin used,name,using the Kaggle Titanic dataset.

## Files
- `train.csv` – Training dataset  
- `test.csv` – Test dataset  
- `titanic.ipynb` – Jupyter Notebook with code
